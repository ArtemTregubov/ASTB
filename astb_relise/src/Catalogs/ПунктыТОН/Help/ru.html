Данный справочник включает в себя информацию о пунктах ТОН, привязанных к номеру ТОНа.

Для отображения списка пунктов ТОН по номеру ТОНа, при первоначальном открытии необходимо настроить поиск по "владельцу", то есть по номеру ТОН. Нажмите "Все действия" - "Настроить список".

В открывшемся окне необходимо в левой части окна курсором выбрать "Владелец", зажать левую кнопку мыши и перетащить этот элемент в правую часть окна. Должно получиться так:

После этого нажмите "Завершить редактирование".

Теперь в справочнике "Пункты ТОН" появилась строка выбора владельца, т.е. номера ТОНа. Т.к. ТОН пока не выбран, справочник пустой.

Для выбора владельца нажмите <...> и выберите пункт ТОН. Если в списке нет нужного пункта ТОН, его можно создать.

После выбора номера ТОНа, в справочнике "Пункты ТОН" отобразятся все пункты ТОН, относящиеся к данному ТОНу. Если в списке нет нужного пункта, его можно создать.
