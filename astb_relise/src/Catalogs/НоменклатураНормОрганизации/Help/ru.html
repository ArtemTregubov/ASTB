Данный справочник предназначен для ввода наименований средств индивидуальной защиты, используемых организацией. В дальнейшем данные из этого справочника будут выбираться при создании норм выдачи.

Для заполнения данного справочника необходимо зайти в список организаций, двойным нажатием выбрать предприятие и выбрать раздел «Номенклатура норм организации».

Для создания новой позиции необходимо нажать кнопку «Создать», вписать наименование и выбрать из списка тот вид СИЗ, который подходит для нового элемента. Чтобы сохранить изменения, нажмите кнопку «Записать и закрыть».
