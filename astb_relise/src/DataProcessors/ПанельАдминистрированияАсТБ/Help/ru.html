1.          Инструкция по настройке терминала сбора данных в АСТБ

Терминал сбора данных (ТСБ) используется для передачи штрих кодов в АСТБ.
После установки ТСБ на сервере «ts-astb» (делается это по заявке на support), запустите программу АСТБ. В разделе «Администрирование», пункт «Настройка данных для учета СИЗ», прокрутите страничку вниз до заголовка «Торговое оборудование». Поставьте флажок «Использовать подключаемое оборудование» и нажмите кнопку «Подключаемое оборудование».

Выберите пункт «Терминалы сбора данных». Далее нажмите кнопку «Создать». Во всплывающем окне выберите обработчик драйвера, согласно модели ТСБ. Наименование будет проставлено автоматически.

Примечание: если в списке нет подходящей модели, попробуйте выбрать «Сканкод: Терминалы сбора данных».

Нажмите кнопку «Сохранить», а затем «Настроить».

В открывшемся окне, на вопрос «Установить драйвер?», ответ – «Нет».

Во всплывающем окне необходимо выбрать модель, порт, тип связи и скорость, согласно модели ТСБ, поставить галочку «Выбирать источник загрузки».

Нажмите «Записать и закрыть».

Снова нажмите на кнопку «Настроить», на вопрос «Установить драйвер?», ответ – «Да».

Работа с АСТБ переходит в работу с мастером установки ТСБ. Далее установите драйвера, согласно модели ТСБ.

Для работы с терминалом необходимо ознакомиться с инструкцией по эксплуатации.

После установки драйверов ТСБ проверьте правильность работы. Если в ТСД хранятся штрих коды напечатанных этикеток, список можно посмотреть, зайдя в раздел «Учет чистки СИЗ», «Документы возврата из чистки». При создании документа нажмите на знак сканера.
