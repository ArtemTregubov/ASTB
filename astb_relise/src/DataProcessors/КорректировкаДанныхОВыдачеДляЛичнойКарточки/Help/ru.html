Корректировка данных о выдаче для личной карточки.

Обработка позволяет, при необходимости, убрать из личной карточки сотрудника ненужные записи. 
Такая возможность может пригодиться, например, в случае ошибочного внесения информации о ранее выданных сотрудникам СИЗ на этапе внедрения проекта. 
Заполняем поля «организация» и «сотрудник», нажимаем «Прочитать записи»:


Табличная часть заполнится всеми выдачами выбранного сотрудника:


Для «исключения» выдачи из личной карточки выделяем галочками те выдачи, которые хотим исключить, нажимаем «Сохранить изменения»:




После этого информация о выделенных выдачах пропадет из личной карточки.
Если по отмеченным выдачам были списания, списания также не попадут в личную карточку.
Для отмены внесенных изменений точно так же заходим в обработку, выбираем организацию и сотрудника. Снимаем галочки там, где необходимо, нажимаем «Сохранить изменения».
