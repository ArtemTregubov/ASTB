Данный регистр содержит сведения обо всех уточненных основаниях норм выдачи.

Для того, чтобы добавить уточненное основание, нажмите кнопку "Создать".

Далее необходимо выбрать норму выдачи, для которой вводится уточненное основание и должность, к которой относится пункт ТОН.

После введения нормы выдачи и должности необходимо выбрать основание, т.е. пункт ТОН.

Для этого нажмите кнопку <...>, откроется справочник "Пункты ТОН". При первоначальном открытии необходимо настроить поиск по "владельцу", то есть по номеру ТОН. Нажмите "Все действия" - "Настроить список".

В открывшемся окне необходимо в левой части окна курсором выбрать "Владелец", зажать левую кнопку мыши и перетащить этот элемент в правую часть окна. Должно получиться так:

После этого нажмите "Завершить редактирование".

Теперь в справочнике "Пункты ТОН" появилась строка выбора владельца, т.е. номера ТОНа. Т.к. ТОН пока не выбран, справочник пустой.

Для выбора владельца нажмите <...> и выберите пункт ТОН. Если в списке нет нужного пункта ТОН, его можно создать.

После выбора номера ТОНа, в справочнике "Пункты ТОН" отобразятся все пункты ТОН, относящиеся к данному ТОНу. Если в списке нет нужного пункта, его можно создать.

После выбора пункта ТОН по владельцу, в окне "Уточненные основания норм выдачи" нажмите "Записать и закрыть".
