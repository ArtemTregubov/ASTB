Прежде, чем создавать места хранения СИЗ, необходимо задать шаблон, по которому будет происходить нумерация мест хранения. Для создания шаблона необходимо зайти в справочник организаций, выбрать созданную фирму, пункт «Настройка шаблона мест хранения СИЗ». Добавьте новый шаблон, нажав кнопку «Создать».

В открывшемся окне можно изменить количество элементов адреса, сняв или поставив флажок у значения «Использовать».

Также можно задать название самих ячеек в поле «Представление», например, «Ряд» вместо «Линия».

В ячейке «Представление пустого значения» можно указать любой символ, который будет означать, что данный элемент не учитывается при адресации. К примеру, во всех помещениях используется понятие секция, а в каком-то одном не используется, тогда при нумерации помещения без секции не нужно заполнять номер секции, а вместо пустого значения проставить «*».

Также можно выбрать разделитель, для удобства прочтения адреса на этикетке. После введения всех данных будет представлен пример адресации места хранения СИЗ.

Когда все необходимые поля будут заполнены, нажмите кнопку «Записать и закрыть».

Примечание: один шаблон адресации мест хранения СИЗ настраивается на всю организацию.
