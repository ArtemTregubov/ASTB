Данный отчет отображает нормы выдачи СИЗ для выбранных сотрудников.

Для формирования отчета необходимо зайти в раздел «Нормы», «Панель отчетов по нормам». В открывшемся окне выбрать отчет «Нормы по сотруднику».

Укажите дату анализа и организацию. Проставьте флажок напротив параметра «Сотрудник» и нажмите на кнопку <...>.

С помощью подбора определите список сотрудников для формирования отчета.

При необходимости определите вид выдачи (коллективная или персональная выдача), заранее проставив флажок напротив указанного параметра. Затем нажмите на кнопку «Сформировать».

По указанным параметрам будет сформирован отчет.

Чтобы распечатать сформированный отчет, нажмите кнопку   на командной панели отчета. Сохранить отчет можно с помощью кнопки .
