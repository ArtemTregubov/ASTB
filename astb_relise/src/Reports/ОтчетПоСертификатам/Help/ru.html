Данный отчет позволяет следить за ситуацией с сертификатами номенклатуры.

Для открытия отчета зайдите в раздел "Предприятие" - "Панель отчетов по предприятию" - "Отчет по сертификатам".

Возможные настройки отчета:

"Выводить код номенклатуры", "Выводить код синхронизации номенклатуры", "Выводить артикул номенклатуры" - настройка необходимой информации, в зависимости от того, какие коды использует предприятие.

"Выводить только действующие" - Отобразятся только сертификаты с не истекшим сроком действия или бессрочные.

"Номенклатура (в группе)" - можно выбрать группу номенклатуры.

"Номенклатура.Вид СИЗ" - можно отобрать по виду СИЗ, т.е. только каски, или только брюки.

"Сертификат" - отбор по номеру сертификата.

Как видно на скриншоте, отчет отображает информацию по всей номенклатуре выдачи. В случае отсутствия сертификата появляется надпись "Отсутствует".
