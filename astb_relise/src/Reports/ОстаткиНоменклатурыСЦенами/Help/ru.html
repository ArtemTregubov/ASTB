Данный отчет можно сформировать только при наличии специального доступа. Он отражает данные по остаткам номенклатуры с ценами, помогает определить полную стоимость товаров, находящихся на складе.

На странице отчета укажите дату и организацию. В поле "быстрых" пользовательских настроек будет выведен перечень возможных параметров для отбора данных при формировании отчета:

1.       Флажок «Отображать характеристики» - ставится, если необходимо узнать количество и цену изделий с учетом размеров. Если флажок не будет проставлен, отчет покажет общее количество номенклатуры на складе без уточнения размеров.

2.       Склад – возможен отбор данных по определенному складу выдачи.

3.       Номенклатура – также возможен отбор данных по указанной номенклатуре выдачи.

Задайте нужные значения для выбранных параметров и нажмите на кнопку «Сформировать».

После этого будет выведен отчет с учетом стоимости номенклатуры.
