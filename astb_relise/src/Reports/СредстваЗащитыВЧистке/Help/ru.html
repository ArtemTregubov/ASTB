1.     Ведомость по СИЗ в чистке

Данный отчет выводит информацию по перемещениям спецодежды в процессе чистки. Он показывает, сколько изделий было передано в чистку и сколько из них было возвращено на склад.

Чтобы сформировать отчет, зайдите в раздел «Учет чистки СИЗ», «Панель отчетов по учету чистки СИЗ», и выберите «Ведомость по СИЗ в чистке».

Отчет может быть сформирован по всем СИЗ, которые отгружаются в чистку, а также по отдельной номенклатуре выдачи.

Нажмите на кнопку «Сформировать», после чего будет выведен отчет.

В отчете будут отражены следующие данные:

1.       ФИО владельца переданного в чистку СИЗ.

2.       Наименование и характеристика номенклатуры выдачи.

3.       Приход – количество изделий, переданных в чистку.

4.       Расход – количество изделий, возвращенных из чистки на склад.

5.       Конечный остаток – количество изделий, которые находятся в чистке на момент формирования отчета.

Чтобы распечатать сформированный отчет, нажмите кнопку   на командной панели отчета. Сохранить отчет можно с помощью кнопки .

 

2.     Остатки СИЗ в чистке

Данный отчет выводит информацию по тем СИЗ, которые находятся в чистке на период формирования отчета. Отчет может быть сформирован по всем видам СИЗ, а также по отдельной номенклатуре выдачи.

Чтобы сформировать отчет, зайдите в раздел «Учет чистки СИЗ», «Панель отчетов по учету чистки СИЗ», и выберите «Ведомость по СИЗ в чистке».

Задайте необходимые параметры для построения отчета и нажмите на кнопку «Сформировать».

В отчете будут отражены следующие данные:

1.       ФИО владельца переданного в чистку СИЗ.

2.       Наименование и характеристика номенклатуры выдачи.

3.       Конечный остаток – количество изделий, которые находятся в чистке на момент формирования отчета.

4.       Вес  изделий, находящихся в чистке.
