Данный отчет отражает потребность сотрудников в СИЗ на заданный период.

Для формирования отчета необходимо зайти в раздел «Обеспечение сотрудников», «Панель отчетов по обеспечению сотрудников». В открывшемся окне необходимо выбрать отчет «Потребность выдачи СИЗ».

Для формирования отчета необходимо указать следующие данные:

1.       Период отчета – устанавливается для отбора данных по выданным СИЗ. Он может быть произвольным (установка начала и конца периода) или фиксированным (день, неделя, декада, месяц, квартал, полугодие и год).

2.       Организация.

3.       Приоритет:

·         Максимум – приоритетными будут считаться изделия с наибольшей ценой.

·         По среднему – расчет стоимости будет происходить по усредненной цене за номенклатуру.

·         Минимум – приоритетными будут изделия с наименьшей ценой.

·         По основному – будут учитываться приоритеты, проставленные в документе соответствий номенклатуры выдачи.

4.       Флажок «Учитывать не выданные в прошлом периоде СИЗ» - устанавливается, если необходимо вывести в отчет данные по тем СИЗ, которые не были выданы.

5.       Флажок «Подразделение» – ставится в случае, когда необходимо произвести отбор данных по одному или нескольким подразделениям.

6.       Флажок «Вид выдачи СИЗ» – может быть реализован отбор по выбранному виду выдачи (персональная или коллективная).

Построение отчета может быть произведено на прошедший, текущий или будущий период. В любом случае потребность будет рассчитана относительно даты формирования отчета.

4.            Формирование отчета

После того, как все нужные параметры будут заданы, нажмите на кнопку «Сформировать».

Примечание: график обеспечения будет доступен для использования только после установки алгоритма формирования потребности.

Далее будет сформирован отчет, отражающий потребность каждого сотрудника организации, с учетом стоимости номенклатуры и антропометрии сотрудников.

Примечание: для того, чтобы нужное антропометрическое значение появилось в отчете, его нужно указать в справочнике номенклатур; чтобы выводилось значение роста, необходимо проставить флажок «Использовать рост».
