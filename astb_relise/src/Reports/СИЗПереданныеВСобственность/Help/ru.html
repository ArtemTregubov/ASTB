Данный отчет составляется на основании документов «Переход прав собственности», может быть использован для сверки с выставленными клиенту счетами.

Чтобы сформировать отчет, зайдите в раздел «Выдача», «Панель отчетов по выдаче», и выберите отчет «СИЗ переданные организациям».

В панели «быстрых» пользовательских настроек задайте нужные параметры для формирования отчета. Чтобы просмотреть полный список СИЗ, переданных организации, не проставляя флажки и значения параметров, нажмите на кнопку «Сформировать».

После этого будет сформирован отчет.

В отчете будут отражены следующие данные:

1.       Наименование организации.

2.       Наименование МВЗ.

3.       Наименования и характеристики переданной номенклатуры.

4.       Количество и стоимость переданных СИЗ.
