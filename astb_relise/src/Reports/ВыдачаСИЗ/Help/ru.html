1.     Выдача СИЗ по номенклатуре выдачи

Данный отчет отображает данные по выдаче номенклатуры за установленные временные периоды. Он может быть полезен для составления статистических графиков по выдаваемой номенклатуре.

Для формирования отчета необходимо зайти в раздел «Выдача», «Панель отчетов по выдаче». В открывшемся окне выбрать отчет «Выдача СИЗ по номенклатуре выдачи».

Панель «быстрых» настроек будет содержать следующие параметры отбора:

1.       Период – устанавливается для отбора данных по выданной номенклатуре. Он может быть произвольным (установка начала и конца периода) или фиксированным (день, неделя, декада, месяц, квартал, полугодие и год).

2.       Отображать характеристику номенклатуры – Да/Нет – выбор влияет на то, будут ли выводиться данные по характеристике выданной номенклатуры.

3.       Тип периода – устанавливается для разделения выведенных данных по выбранному периоду (день, неделя, декада, месяц, квартал, полугодие и год). Например, если информация выводится за год, ее можно разбить по месяцам.

4.       Номенклатура – может быть реализован отбор данных по заданной номенклатуре выдачи.

Задайте необходимые значения для выбранных параметров и нажмите кнопку «Сформировать».

Будет сформирован отчет по выданной номенклатуре.

В отчете будут отображены:

1.       Информация по номенклатуре выдачи: наименование, характеристики, код синхронизации (ORA) и цена.

2.       Количество выданных СИЗ по месяцам, т.к. для параметра «Тип периода» было задано значение «Месяц».

3.       Количество выданных СИЗ за выбранный период (на примере – год).

4.        Стоимость указанных наименований.

Чтобы распечатать сформированный отчет, нажмите кнопку   на командной панели отчета. Сохранить отчет можно с помощью кнопки .
