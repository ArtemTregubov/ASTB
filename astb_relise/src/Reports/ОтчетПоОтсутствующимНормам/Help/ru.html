Данный отчет отражает информацию о должностях в указанных подразделениях, для которых отсутствуют нормы выдачи.

Для формирования отчета необходимо зайти в раздел «Нормы», «Панель отчетов по нормам». В открывшемся окне выбрать «Отчет по отсутствующим нормам».

1.            По штатному расписанию

В открывшемся окне укажите период и организацию. Затем в поле «Источник данных» выберите пункт «Штатное расписание». В этом случае отчет будет выводить все должности, для которых нет принятых норм, даже если их не занимают сотрудники. При необходимости можно указать подразделение, заранее проставив флажок напротив используемого параметра. Затем нажмите на кнопку «Сформировать».

После этого будет сформирован отчет. В нем будут указаны те должности, по которым в организации нет принятых норм, без учета занятых рабочих мест.

Примечание: если для формирования отчета было указано подразделение, в отчете будут также отражены дочерние подразделения.

2.            По занятым рабочим местам

Чтобы сформировать отчет по тем должностям, которые занимают сотрудники организации, в поле «Источник данных» укажите «Занятые рабочие места». При необходимости можно указать подразделение, заранее проставив флажок напротив используемого параметра. Затем нажмите на кнопку «Сформировать».

В соответствии с указанными параметрами будет сформирован отчет.

Примечание: если для формирования отчета было указано подразделение, в отчете будут также отражены дочерние подразделения.

Чтобы распечатать сформированный отчет, нажмите кнопку   на командной панели отчета. Сохранить отчет можно с помощью кнопки .
