Данный отчет отображает количество номенклатуры, которая понадобится для покрытия потребности по всему штатному расписанию (вне зависимости от того, сколько ставок занято на предприятии) на заданный период, с учетом стоимости. Отчет формируется на основе документа «Установка соответствий номенклатуры выдачи» для каждой должности из ШР.

Чтобы сформировать отчет, зайдите в раздел «Обеспечение сотрудников», «Панель отчетов по обеспечению сотрудников», и выберите отчет «Нормы по подразделению по ШР».

Для формирования отчета необходимо указать следующие данные:

1.       Организация.

2.       Количество месяцев в периоде – если необходимо задать год, значение данного поля 12, для полугодия – 6 и т.д.

3.       Поставщик – название организации поставщика СИЗ и СОД из справочника «Контрагенты».

4.       Приоритет:

·         Основной – учитывается номенклатура с первым приоритетом в документе соответствия номенклатуры выдачи.

·         Максимальная цена – приоритетными будут считаться изделия с наибольшей ценой.

·         Минимальная цена – приоритетными будут изделия с наименьшей ценой.

5.       Флажок «Подразделение»  – ставится в случае, когда необходимо произвести отбор данных по одному или нескольким подразделениям.

6.       Флажок «Номенклатура нормы организации» - ставится в случае, когда необходимо произвести отбор данных по выбранной номенклатуре нормы.

После того, как все необходимые параметры будут заданы, нажмите на кнопку «Сформировать».

После этого будет сформирован отчет.

Примечание: красным цветом будут отмечены те строки, для которых не указана цена.

В отчете будет отражено:

1.       Должности из штатного расписания и подразделения, к которым они относятся.

2.       Принятые нормы для указанных должностей.

3.       Номенклатура выдачи на основании документа соответствий номенклатуры выдачи с указанием приоритетов и цены.

4.       Количество ставок для указанных должностей в штатном расписании.

5.       Количество на период – количество номенклатуры, необходимое для покрытия потребности за выбранный период.

6.       Общая стоимость.
