Данный отчет предусмотрен для просмотра как существующих, так и не установленных соответствий номенклатуры выдачи. Кроме того, в нем указываются строки уточненных соответствий.

Для формирования отчета необходимо зайти в раздел «Снабжение», «Панель отчетов по снабжению». В открывшемся окне необходимо выбрать отчет «Соответствия номенклатуры выдачи».

В открывшемся окне укажите организацию. Панель «быстрых» настроек содержит следующие параметры отбора:

1.       Поставщик – отбор данных может быть произведен с учетом выбранного поставщика.

2.       Вид выдачи СИЗ – отбор также может быть реализован по указанному виду выдачи СИЗ (персональная или коллективная).

3.       Комментарий:

·         Нет маппинга – в отчет будут выведены строки с номенклатурой нормы, для которой не было установлено соответствующей номенклатуры выдачи.

·         Нет нормы выдачи – отчет заполнится соответствиями, для которых не было установлено норм.

·         Есть соответствие – в отчете отразятся строки соответствий номенклатуры нормы и выдачи без учета уточнений.

·         Уточненный маппинг – будут выведены строки уточненного соответствия номенклатуры выдачи.

Примечание: может быть установлено несколько значений поля «Комментарий».

Примечание: маппинг – название, используемое для краткого обозначения соответствий номенклатуры выдачи.

После настройки параметров отбора данных нажмите на кнопку «Сформировать».

В соответствии с указанными параметрами будет сформирован отчет.
