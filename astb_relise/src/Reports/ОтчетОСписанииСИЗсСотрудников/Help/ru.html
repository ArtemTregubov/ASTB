Отчет предназначен для просмотра статистики по списанным СИЗ.

Для формирования отчета необходимо зайти в раздел «Обеспечение сотрудников», «Панель отчетов по обеспечению сотрудников». В открывшемся окне необходимо выбрать отчет «Списание СИЗ с сотрудников».

Для формирования отчета необходимо указать следующие данные:

  1. Период – устанавливается для отбора данных по списанным СИЗ. Он может быть произвольным (установка начала и конца периода) или фиксированным (день, неделя, декада, месяц, квартал, полугодие и год).
  2. Организация.
  3. Флажок «Подразделение (в группе)» – устанавливается, если необходимо произвести отбор по выбранному подразделению.
  4. Флажок «Номенклатура нормы» – ставится в случае, когда необходим отбор данных по заданной номенклатуре нормы.
  5. Флажок «Причина списания» – может быть реализован отбор по выбранной причине списания СИЗ (брак, износ, порча, пропажа, увольнение или окончание срока носки).

 

После того, как все необходимые параметры будут заданы, нажмите на кнопку «Сформировать».

Затем будет сформирован отчет.

В отчете будут отражены:

1.       Данные по сотруднику, с которого производилось списание.

2.       Наименование номенклатуры нормы.

3.       Данные по номенклатуре выдачи: наименование и характеристика.

4.       Причина списания.

5.       Даты выдачи и списания номенклатуры.

6.       Даты окончания носки списанной номенклатуры.
