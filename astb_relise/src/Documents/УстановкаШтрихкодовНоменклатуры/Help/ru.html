Для возможности ведения учета документов передачи в чистку и возврата из чистки в программе АСТБ необходимо сформировать штрихкоды для всех видов СИЗ, которые планируется передавать в чистку.

1. Формирование штрихкода

Чтобы сгенерировать штрихкод для этикеток, в АСТБ используется значение префикса штрихкода номенклатуры, которое должно быть задано в карточке организации. 
Это необходимо для того, чтобы при ведении учета в одной базе нескольких юридических лиц нумерация штрих кодов не была сквозной и была возможность визуального разделения.
Указать префикс штрихкода номенклатуры необходимо через раздел «Предприятие»-справочник- «Организации»  открыть карточку организации.

В блоке «Учет чистки» указать префикс штрихкода номенклатуры (должен состоять их трех цифр), на рис.1 указан пример– «852»)

2. Настройка номенклатуры выдачи

Необходимо определить список видов СИЗ, которые планируется передавать в чистку, и проставить у соответствующего вида СИЗ определенный признак (использование штрих кода). Для этого необходимо зайти в раздел «Предприятие», справочник «Номенклатура выдачи».
В открывшемся справочнике необходимо выбрать номенклатуру выдачи, которую планируется отдавать в чистку.
При этом необходимо понимать из скольких элементов состоит выбранная номенклатура.
Например, костюм состоит из двух составляющих (куртка и брюки), в данном случае этикетка должна наносится на каждую единицу.
Для настройки комплектности номенклатуры необходимо в самой номенклатуре -на вкладке «Данные для учета СИЗ» проставить  Комплект     (рис.3)



* «Использовать штрихкод» в номенклатуре комплекта не включается

-на вкладке «Комплектующие» указать составные части этого комплекта (рис.4)



В карточке номенклатуры указанных комплектующих необходимо поставить  «Использовать штрихкод» (рис.5)



Если номенклатура не состоит из нескольких элементов (например Жилет»), настройка комплектности не требуется, устанавливается только  «Использовать штрихкод»

В этом же блоке «Обслуживание» можно указать допустимое количество циклов чистки, если в карточке организации включена соответствующая опция (рис.6)

А затем сохраните данные, нажав кнопку «Записать и закрыть».



3. Создание штрихкода номенклатуры

3.1. Для установки штрихкодов на ранее выданные СИЗ необходимо воспользоваться документом «Установка штрихкодов номенклатуры» в разделе «Обслуживание»
Для заполнения документа необходимо указать:
-Организацию
-сотрудника
-вид выдачи (Персональная/произвольная)
Далее нажать на кн. «Заполнить»--«По всем выданным СИЗ»

Табличная часть будет заполнена выданной номенклатурой, для которой значения штрих кодов не были установлены ранее (этикетки не были распечатаны)   (рис.7)




* документ отберет только те средства защиты, для которых был установлен признак «Использовать штрихкод»

Далее для установки штрихкода необходимо нажать «Заполнить»--«Пустые штрихкоды» (рис.8,9)







Из документа доступен сервис печати этикеток по соответствующей кнопке "Этикетки"

Табличная часть будет заполнена всеми выданными товарами с установленными штрих кодами. Отметьте флажками те наименования, для

которых нужно распечатать этикетки. Затем нажмите кнопку «Печать».
