Установка зимних месяцев для работы условий выдачи «Зимой дополнительно» и «Всегда в зимнее время». Используются для создания норм выдачи СИЗ.

Чтобы задать период зимы, зайдите в раздел «Нормы», документ «Установка периода зимы». В открывшемся окне нажмите кнопку «Создать».

В окне «Установка периода зимы (создание)» выберите организацию и нажмите кнопку «Добавить». Период зимы можно проставить как для конкретных подразделений, так и для всей организации. Если нужно присвоить значения начала и конца зимы для отдельных подразделений, добавьте нужное количество строк и введите всю необходимую информацию. Не забудьте проставить флажки для всех используемых периодов. Затем нажмите кнопку «Провести и закрыть».

Если для организации или ее отдельных подразделений уже были установлены периоды зимы, в данной форме нажмите кнопку «Заполнить». Тогда в табличной части будут проставлены все значения зимы, которые были приняты ранее.

Командная панель формы «Приказ по нормам выдачи СИЗ» содержит следующие вспомогательные кнопки:

 – Создать новый элемент копированием предыдущего;

 – Удалить текущий элемент;

 – Переместить текущий элемент вверх;

 – Переместить текущий элемент вниз;

  – Установить флажки;

 – Снять флажки.
