Произвольная выдача

На пункт выдачи спецодежды могут приходить сотрудники и посетители, которые по каким-либо причинам должны получить спецодежду вне потребности, то есть не по нормам, принятым в организации. В подобном случае для получения спецодежды сотрудник обязан обратиться в отдел по технике безопасности, где для него составят заявку для произвольной выдачи. Однако в некоторых случаях составления заявки на произвольную выдачу может производиться сотрудниками склада.

1.     Формирование заявки

Чтобы создать заявку, зайдите в раздел «Выдача», документ «Заявки на произвольную выдачу СИЗ». Чтобы добавить новый документ, нажмите кнопку «Создать».

В открывшемся окне выберите: организацию; склад, с которого будет производиться выдача; МВЗ, которое будет нести материальную ответственность за произвольную выдачу; ФИО заказчика (сотрудник отдела ТБ); ФИО сотрудника, которому будут выдаваться СИЗ.

Заполнение табличной части может осуществляться двумя способами: с указанием номенклатуры нормы или же с указанием номенклатуры выдачи.

1.            Заполнение заявки по номенклатуре нормы

Сотрудник отдела ТБ может составлять заявки на произвольную выдачу, указывая только номенклатуру нормы, которую необходимо выдать, без указания соответствующей номенклатуры выдачи и ее характеристик.

Чтобы занести номенклатуру в список, нажмите кнопку «Добавить». В появившейся строке двойным нажатием выделите поле для добавления номенклатуры нормы, затем нажмите кнопку <...> и выберите нужное наименование из справочника. Затем определите количество выдаваемой номенклатуры.

Добавьте необходимые строки с указанием номенклатуры нормы и ее количества для выдачи. Затем нажмите на кнопку «Записать и закрыть».

Примечание: после проведения заявка на произвольную выдачу может быть распечатана. Для этого нажмите на кнопку . В открывшемся окне нажмите кнопку «Печать».

2.            Заполнение заявки по номенклатуре выдачи

Работникам склада может быть удобнее заполнять заявку на произвольную выдачу с учетом номенклатуры выдачи, так как они могут отслеживать наличие товара на складе выдачи.

Чтобы добавить номенклатуру в список, нажмите на кнопку «Подбор номенклатуры выдачи».

Откроется окно подбора номенклатуры. В нем нажатием выберите папку «Номенклатура выдачи» и проставьте флажок «Только в наличии на складе» напротив нужного склада. Затем выберите нужную номенклатуру выдачи, в окне ниже выделите ее характеристику и, удерживая левую кнопку мыши, перетащите ее в окно данных для переноса в документ. Далее укажите количество номенклатуры для выдачи. Затем нажмите кнопку «Перенести в документ».

Выбранная номенклатура будет проставлена в табличной части заявки.

Примечание: в данном случае определять номенклатуру нормы не требуется.

Когда список будет сформирован, нажмите на кнопку «Записать и закрыть».

Примечание: после проведения заявка на произвольную выдачу может быть распечатана. Для этого нажмите на кнопку . В открывшемся окне нажмите кнопку «Печать».

2.     Проведение выдачи

Произвольная выдача осуществляется на основании ранее созданной заявки.

1.            Выдача по заявке с указанием номенклатуры нормы

Чтобы провести выдачу СИЗ на основании заявки, в которой была указана номенклатура нормы, зайдите на страницу нужной заявки и нажмите кнопку «Создать на основании», «Выдача средств защиты сотруднику».

Откроется окно «Выдача средств защиты сотруднику (создание)». В документе будут заполнены необходимые данные по организации, складу, МВЗ, заказчику и сотруднику. В табличной части будет указана номенклатура нормы, по которой должна быть произведена выдача. Чтобы подобрать номенклатуру выдачи, нажатием выделите нужную строку и нажмите кнопку «Заполнить», пункт «Подобрать номенклатуру в строке».

Откроется окно подбора номенклатуры. В левой части таблицы будет отражена номенклатура для выделенной строки документа выдачи. Развернув представленный иерархический список, можно увидеть все существующие размеры указанной номенклатуры. Зеленым цветом будут окрашены те характеристики, которые находятся в наличие на складе, в столбце «Остаток» будет отражено их количество.

Руководствуясь антропометрическими данными сотрудника и остатками на складе, выберите нужную номенклатуру, проставьте флажок, тем самым отмечая строку для переноса в документ. Затем нажмите кнопку «Перенести в документ».

Повторите описанные выше действия для всех строк табличной части. Когда список будет сформирован, нажмите на кнопку «Провести и закрыть».

Чтобы распечатать документ, подтверждающий выдачу, нажмите на кнопку «Печать», пункт «Ведомость на выдачу СИЗ». В открывшемся окне снова нажмите «Печать».

2.            Выдача по заявке с указанием номенклатуры выдачи

Чтобы провести выдачу СИЗ на основании заявки, в которой была указана номенклатура выдачи, зайдите на страницу нужной заявки и нажмите кнопку «Создать на основании», «Выдача средств защиты сотруднику».

Откроется окно «Выдача средств защиты сотруднику (создание)». В документе будут заполнены необходимые данные по организации, складу, МВЗ, заказчику и сотруднику. В табличной части будет указана номенклатура выдачи, ее характеристика и количество для выдачи. Данный документ не требует никаких изменений. После проведения выдачи, нажмите на кнопку «Провести и закрыть».

Примечание: чтобы распечатать документ, подтверждающий выдачу, нажмите на кнопку «Печать», пункт «Ведомость на выдачу СИЗ». В открывшемся окне снова нажмите «Печать».
