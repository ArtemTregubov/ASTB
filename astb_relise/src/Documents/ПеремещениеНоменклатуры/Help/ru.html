Существует несколько способов формирования документа по перемещению номенклатуры с одного склада на другой.

1.     Перемещение по остаткам на складе-отправителе

Для перемещения СИЗ зайдите в раздел «Склад», документ «Перемещение номенклатуры». Нажмите кнопку «Создать», чтобы добавить новый документ. В открывшемся окне выберите организацию, склад, с которого нужно переместить позиции, и склад, на который производится перемещение. Затем нажмите кнопку «Заполнить» и выберите пункт «Заполнить по остаткам на складе отправителе».

После этого табличная часть будет заполнена всеми товарами, которые в данный момент содержаться на складе-отправителе (на примере выше – Склад №1). С помощью кнопок командной панели отберите нужную номенклатуру:

 – Создать новый элемент копированием предыдущего;

 – Удалить текущий элемент;

 – Переместить текущий элемент вверх;

 – Переместить текущий элемент вниз.

Когда список будет сформирован, нажмите кнопку «Провести и закрыть».

2.     Перемещение по коллективной выдаче

Если один из складов организации был назначен для коллективной выдачи и на него нужно перевезти соответствующую номенклатуру, можно воспользоваться перемещением номенклатуры по коллективной выдаче. Для этого зайдите в раздел «Склад», документ «Перемещение номенклатуры». Нажмите кнопку «Создать», чтобы добавить новый документ. В открывшемся окне выберите организацию, склад, с которого нужно переместить позиции, и склад, на который производится перемещение. Склад-получатель должен быть назначен для коллективной выдачи. Затем нажмите кнопку «Заполнить» и выберите пункт «Заполнить по коллективной выдаче».

После этого табличная часть заполнится номенклатурой для коллективной выдачи (СИЗ с короткими сроками носки). С помощью кнопок командной панели отберите нужную номенклатуру:

 – Создать новый элемент копированием предыдущего;

 – Удалить текущий элемент;

 – Переместить текущий элемент вверх;

 – Переместить текущий элемент вниз.

После того, как документ будет сформирован, нажмите кнопку «Провести и закрыть».

 

3.     Ручной подбор номенклатуры

Для осуществления ручного подбора номенклатуры для перемещения зайдите в раздел «Склад», документ «Перемещение номенклатуры». Нажмите кнопку «Создать», чтобы добавить новый документ. В открывшемся окне выберите организацию, склад, с которого нужно переместить позиции, и склад, на который производится перемещение. Затем нажмите кнопку «Заполнить» и выберите пункт «Подбор номенклатуры выдачи».

Откроется соответствующее окно. В нем нажатием выберите папку «Номенклатура выдачи» и проставьте флажок «Только в наличии на складе» напротив нужного склада. Затем выберите номенклатуру для перемещения на другой склад, ее характеристику и, удерживая левую кнопку мыши, перетащите ее в окно данных для переноса в документ. После переноса укажите количество перемещаемых товаров. Затем нажмите кнопку «Перенести в документ».

Затем в окне «Списание номенклатуры (создание)» нажмите кнопку «Провести и закрыть».
