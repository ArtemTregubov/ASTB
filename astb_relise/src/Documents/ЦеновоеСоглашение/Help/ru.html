Для ручного ввода документа зайдите в раздел «Снабжение», пункт «Ценовое соглашение», и нажмите кнопку «Создать».

В открывшемся окне выберите организацию и компанию поставщика. Нажмите кнопку «Добавить», после чего появится строка для ввода данных, а именно номенклатуры выдачи и соответствующей ей цены. В поле для заполнения номенклатуры выдачи нажмите кнопку <...> – появится окно справочника «Номенклатура выдачи». Из списка слева выберите нужного поставщика, а из табличной части справа – нужное наименование. Затем нажмите кнопку «Выбрать», после чего выбранная номенклатура появится в табличной части окна создания ценового соглашения. Укажите цену номенклатуры.

Последовательно заполните табличную часть по всей представленной номенклатуре выдачи, а затем нажмите кнопку «Провести и закрыть» для сохранения введенных данных.

Командная панель данной формы содержит следующие кнопки:

 – Создать новый элемент копированием предыдущего;

 – Удалить текущий элемент;

 – Переместить текущий элемент вверх;

 – Переместить текущий элемент вниз.
